NLP · Defense

Insider Threat Detection & Triage System

Developed NLP-based triage and risk-scoring methodologies for insider threat reports at the Department of Defense. Engineered automated data labeling and annotation schemas to prioritize threat reports by severity, confidence scores, and behavioral indicators — reducing analyst workload and improving response time. Designed for integration into Army Vantage, the Army's enterprise AI platform.

Computer Vision · Defense

IR Image Captioning & Visual Question Answering

Built an end-to-end pipeline for generating natural language captions and answering questions about infrared (FLIR) imagery — bridging the gap between mature civilian RGB-based VQA systems and the underserved military thermal imaging domain. Combines YOLO-based object detection with spatial action inference and transformer-based language generation.

Systems · Networking

Network Protocol Implementations

Built a router, switch, and TCP/UDP client-server stack from scratch in Python using raw sockets. The router implements OSPF-style link-state routing with Dijkstra's algorithm for shortest-path forwarding table computation and longest prefix match for IP packet forwarding.
